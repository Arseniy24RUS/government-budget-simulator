# Roadmap

This roadmap describes practical research-readiness and classroom-readiness improvements. It is not a release promise.

## Near Term

- Keep methodology documentation aligned with the actual model logic.
- Improve issue/PR templates for bugs, feature requests, and methodology questions.
- Add clearer test coverage for README links, local assets, and i18n regressions.
- Review third-party asset usage, especially official symbols and logos.
- Add small instructor-facing examples for interpreting final KPI/support combinations.

## Medium Term

- Add a deterministic seed or scenario/event selection mode for reproducible classroom experiments.
- Extract scenario, event, KPI, group, and policy configuration from `index.html` into structured data files.
- Add validation tests for budget totals, policy cost constraints, KPI bounds, and public-support calculations.
- Provide instructor notes for adapting the simulator to another region or course.
- Improve leaderboard data policy notes and optional offline mode.

## Longer Term

- Add scenario comparison exports for classroom debriefs.
- Support multiple policy-calibration presets.
- Add accessibility review and keyboard-focused testing.
- Add a transparent changelog for methodology-affecting changes.
- Consider a lightweight research protocol template for classroom studies.

## Русский

Эта дорожная карта описывает улучшения для исследовательской и учебной готовности проекта. Она не является обещанием релиза.

Ближайшие задачи: синхронизировать методологию с логикой модели, улучшить шаблоны issue/PR, добавить проверки ссылок и локальных ресурсов, проверить сторонние активы и подготовить примеры интерпретации KPI/поддержки.

Среднесрочно: добавить детерминированный seed или ручной выбор сценария/событий, вынести конфигурацию модели из `index.html`, покрыть тестами бюджетные ограничения и расчет поддержки, подготовить инструкции для адаптации курса и уточнить политику данных лидерборда.

Долгосрочно: добавить экспорт сравнений сценариев, пресеты калибровки, аудит доступности, журнал изменений методологии и шаблон исследовательского протокола для учебных исследований.
